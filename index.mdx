WinSux is a one-click PowerShell script suite that transforms a fresh Windows 10/11 install into a lean, high-performance system. It installs essential software, removes bloatware, configures security settings, installs your GPU drivers with debloating, and optimizes power settings — all automatically.

## Get up and running

<Steps>
  <Step title="Open an elevated PowerShell window">
    Right-click **Start**, select **Terminal (Admin)** or **Windows PowerShell (Admin)**. WinSux requires Administrator privileges.
  </Step>
  <Step title="Run the one-liner">
    Paste this command and press Enter:
    ```powershell
    iwr https://github.com/FR33THYFR33THY/WinSux/raw/refs/heads/main/WinSux.ps1 -useb | iex
    ```
  </Step>
  <Step title="Follow the prompts">
    WinSux runs automatically. When prompted to select your GPU brand for driver installation, choose NVIDIA, AMD, Intel, or skip.
  </Step>
  <Step title="Let it restart">
    The script reboots into Safe Mode to run DDU (Display Driver Uninstaller), then reboots again to complete final configuration. No input required between restarts.
  </Step>
</Steps>

<Note>
  WinSux requires an active internet connection and Windows 10/11 (Home, Pro, LTSC, IoT, or Server editions). Run it on a fresh Windows install for best results.
</Note>

## What WinSux does

WinSux makes hundreds of targeted changes across three phases:

- **Installs** 7-Zip, Google Chrome, DirectX, and Visual C++ runtimes (2005–2022)
- **Removes** Microsoft Edge, OneDrive, most UWP apps, and unnecessary Windows features
- **Configures** Windows Defender, SmartScreen, UAC, and privacy settings
- **Optimizes** power plan (Ultimate Performance), CPU core parking, and timer resolution
- **Installs** your GPU driver cleanly after DDU wipes old drivers in Safe Mode

<Tip>
  For graphics-specific tweaks beyond driver installation, see the companion [Ultimate](https://github.com/FR33THYFR33THY/Ultimate) repository which handles additional GPU scripts.
</Tip>
